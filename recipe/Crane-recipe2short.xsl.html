3. Convert XML to a hand-editable text form - Crane-xml2txt.xsl

Path: ../xsl/Crane-xml2txt.xsl

Filename: Crane-xml2txt.xsl

https://GitHub.com/CraneSoftwrights/Crane-txt2xml

The output conforms to the input requirements for the Crane-txt2xml environment.

This fragment can be run standalone or can be imported by other shell stylesheets to perform a basic function under the influence of various configurations.

3.1. Invocation parameters and input file

The input file is the XML document to be emitted as text.

escape select="'yes'" as="xs:string" (xsl:param)

Set this to "no" for the result not to be escaped (so no round-tripping)

c:escape select="starts-with('yes',lower-case($escape))" as="xs:boolean" (xsl:variable)

A testable boolean for the invocation parameter

markdown select="'no'" as="xs:string" (xsl:param)

Set this to "no" for the result to use markdown

c:markdown as="xs:boolean" select="starts-with('yes',lower-case($markdown))" (xsl:variable)

A testable boolean for the invocation parameter

use-labels select="'yes'" as="xs:string" (xsl:param)

Set this to "yes" for the result to use the labels instead of names

indent select="'yes'" as="xs:string" (xsl:param)

Set this to "yes" for the result to be indented

c:indent select="starts-with('yes',lower-case($indent))" as="xs:boolean" (xsl:variable)

A testable boolean for the invocation parameter

label-gap select="'yes'" as="xs:string" (xsl:param)

Set this to "yes" for there to be a space after the labels

c:labelGap as="xs:boolean" select="starts-with('yes',lower-case($label-gap))" (xsl:variable)

A testable boolean for the invocation parameter

c:useLabels as="xs:boolean" select="starts-with('yes',lower-case($use-labels))" (xsl:variable)

A testable boolean for the invocation parameter

method="text" (xsl:output)

The serialization must be a simple text file

c:infoLookup as="document-node()" xml:lang="en" (xsl:variable)

Placebo lookup table to be replaced

The expected structure for each has a mandatory element= indicating the name of the element, an optional markdownSymbol= indicating the symbol used in mixed content to represent the element, and the indication of mixed content

c:infoByName match="labelInfo" use="' all ',@name" (xsl:key)

Look up key table for the element/attribute info based on the name

c:info4item(1) as="element()?" (xsl:function)

Look up element/attribute info based on the name

Parameter c:item as="item()"

The element or attribute to look up

c:markdownByName match="elementLabel[@markdownSymbol]" use="'__all__',@markdownSymbol" (xsl:key)

Look up the element info based on the name
