1. Convert UBL XSD to iXML patterns per Crane-txt2xml - Crane-ubl2ixml.xsl

Filename: Crane-ubl2ixml.xsl

Import statement:

  • Convert XSD to iXML patterns per Crane-txt2xml - Crane-xsd2ixml.xsl

https://GitHub.com/CraneSoftwrights/Crane-txt2xml

This automates the generation of iXML from a schema.

1.1. Invocation parameters and input file

The input file is an XSD grammar in the Venetian Blind structure

If the input has the description of mixed content, this is assumed to be simple text.

2. Convert XSD to iXML patterns per Crane-txt2xml - Crane-xsd2ixml.xsl

Path: ../../xsl/Crane-xsd2ixml.xsl

Filename: Crane-xsd2ixml.xsl

https://GitHub.com/CraneSoftwrights/Crane-txt2xml

This automates the generation of iXML from a schema.

2.1. Invocation parameters and input file

The input file is an XSD grammar in the Garden of Eden structure (global names and global types).

If the input has the description of mixed content, this is assumed to be simple text.

method="text" (xsl:output)

An iXML file is simple text.
