# Shell invocation scripts for Crane-txt2ubl

The UBL-specific invocation scripts of the Crane-txt2ubl environment for Linux, macOS, and BSD shells. The Windows batch counterparts are in [`../windows`](../windows).

# Manifest

[`Crane-txt2ubl.sh`](Crane-txt2ubl.sh)
- invoke the Crane-txt2ubl workflow on the text input to create a UBL instance

[`Crane-ubl2txt.sh`](Crane-ubl2txt.sh)
- convert a UBL XML document into a simple text file suitable for editing

[`make-ubl-ixml.sh`](make-ubl-ixml.sh)
- invoke the conversion of the UBL XSD into the iXML grammar

[`README.md`](README.md)

[`test-ubl.sh`](test-ubl.sh)
