# Windows batch invocation scripts for Crane-txt2ubl

The UBL-specific invocation scripts of the Crane-txt2ubl environment for Windows, mirroring the shell scripts in [`../shell`](../shell).

# Manifest

[`Crane-txt2ubl.bat`](Crane-txt2ubl.bat)
- invoke the Crane-txt2ubl workflow on the text input to create a UBL instance

[`Crane-ubl2txt.bat`](Crane-ubl2txt.bat)
- convert a UBL XML document into a simple text file suitable for editing

[`make-ubl-ixml.bat`](make-ubl-ixml.bat)
- invoke the conversion of the UBL XSD into the iXML grammar

[`README.md`](README.md)

[`test-ubl.bat`](test-ubl.bat)
