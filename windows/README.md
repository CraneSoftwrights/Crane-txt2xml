# Windows batch invocation scripts

The vocabulary-independent invocation scripts of the Crane-txt2xml environment for Windows, mirroring the shell scripts in [`../shell`](../shell).

# Manifest

[`Crane-txt2xml.bat`](Crane-txt2xml.bat)
- invoke the Crane-txt2xml workflow converting authored text into XML

[`Crane-xml2txt.bat`](Crane-xml2txt.bat)
- convert XML into a text form suitable for Crane-txt2xml

[`indentXML.bat`](indentXML.bat)
- indent the argument XML file in place
