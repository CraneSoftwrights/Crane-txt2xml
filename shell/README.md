# Shell invocation scripts

The vocabulary-independent invocation scripts of the Crane-txt2xml environment for Linux, macOS, and BSD shells. The Windows batch counterparts are in [`../windows`](../windows).

# Manifest

[`Crane-txt2xml.sh`](Crane-txt2xml.sh)
- invoke the Crane-txt2xml workflow converting authored text into XML

[`Crane-xml2txt.sh`](Crane-xml2txt.sh)
- convert XML into a text form suitable for Crane-txt2xml

[`indentXML.sh`](indentXML.sh)
- indent the argument XML file in place
