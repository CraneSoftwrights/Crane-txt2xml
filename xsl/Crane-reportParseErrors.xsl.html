1. Report parse errors from the iXML output - Crane-reportParseErrors.xsl

Filename: Crane-reportParseErrors.xsl

$Id$

One of the ways in which the process fails is that the grammar is ambiguous and the ixml tool abends with a report.

This stylesheet interprets the ixml report to produce an error report.

This stylesheet is reliant on XPath 3, which means it cannot be used with Saxon HE (Home Edition). The package distribution includes only Saxon HE.

1.1. Invocation parameters and input file

The input file is an XML document with {}ixml as the document element.
